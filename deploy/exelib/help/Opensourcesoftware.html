Open source software


SolarPILOT source code is publicly available at https://github.com/NREL/SolarPILOT. 


SolarPILOT utilizes the following open-source software packages outside of the NREL build chain.


NLOpt -- nonlinear optimization

  • Steven G. Johnson, The NLopt nonlinear-optimization package, http://ab-initio.mit.edu/nlopt
  • M. J. D. Powell, "A direct search optimization method that models the objective and constraint functions by linear interpolation," in Advances in Optimization and Numerical Analysis, eds. S. Gomez and J.-P. Hennart (Kluwer Academic: Dordrecht, 1994), p. 51-67.


LPSolve -- mixed-integer linear programming solver

Description

Open source (Mixed-Integer) Linear Programming system

Language

Multi-platform, pure ANSI C / POSIX source code, Lex/Yacc based parsing

Official name

lp_solve (alternatively lpsolve)

Release data

Version 5.1.0.0 dated 1 May 2004

Co-developers

Michel Berkelaar, Kjell Eikland, Peter Notebaert

Licence terms

GNU LGPL (Lesser General Public Licence)

Citation policy

General references as per LGPL


wxWidgets -- graphical interface widget library

https://www.wxwidgets.org/


RapidXML -- XML parsing library

https://rapidxml.sourceforge.net

Version 1.13

Copyright (C) 2006, 2009 Marcin Kalicinski


Solpos -- high-precision solar position calculator

https://www.nrel.gov/grid/solar-resource/solpos.html

Martin Rymes, National Renewable Energy Laboratory (March 1998)
