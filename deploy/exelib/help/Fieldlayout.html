Page Overview


The Field Layout page provides the execution functions for layout and regeneration of the heliostat field. At the top of the page, you'll find buttons for executing a layout, regenerating geometry with the current layout, timers and progress bars, and the simulation log. Each of these items are covered in more detail below.


The following keyboard shortcuts are enabled for this page:


F5                Generate new layout

Alt+F5                Regenerate current layout

Generate New Layout

- Top of page -


 


Keyboard shortcut:        F5


This button creates a new layout and generates the field geometry, receiver geometry, and other necessary data to define a power tower solar field. When executing a new layout, SolarPILOT collects all of the inputs that you've specified on the other input pages (Climate, Markets, Costs, Plant, Layout Setup, Heliostats, Receivers) and generates an appropriate solar field.


To generate a new layout, SolarPILOT:

  1. Collects inputs and settings  from the input pages.
  2. Pre-processes the weather data or solar irradiance model to generate a set of simulation points for design evaluation.
  3. Generates a list of all possible heliostat positions that fall within the land boundary that you specify. The positions are generated according to the layout rules that you choose on the Layout Setup page.
  4. Simulates the performance of the field - including each possible heliostat in the layout from step (3) - over the set of simulation points generated in step (2). Note that for systems with a large number of heliostats or a large set of simulation points, this step can require significant computation. The progress made by SolarPILOT is updated in the Simulation Timer progress bar and output text. Steps in the process are noted in the Simulation Log.
  5. Sorts all possible heliostats by the performance metric that you specify in the Heliostat selection criteria field on the Layout Setup page. Typically, the heliostats are sorted from highest to lowest on the power that they produce over the simulation set.
  6. Calculates the power delivered by each heliostat at the reference condition by running a single simulation using the reference values specified on the Layout Setup page. The simulation is evaluated at the Design-point DNI value and the Sun Location at Design Point.
  7. Deletes heliostats in order of worst-performing to best-performing while the power delivered at the reference condition is still satisfied. Once the deletion of a heliostat would lead to a delivered power less than the design point (plus thermal losses), the deletion process terminates.


The remaining heliostats form the field layout that is displayed on the Layout Results page.


While running, you can cancel the layout process by clicking the run button (which is displayed with the red cancel option). The layout process will be aborted.


Regenerate Current Layout

- Top of page -



Keyboard shortcut:        Alt+F5


It may be the case that you wish to modify the inputs or settings and apply the changes to an existing layout. The Regenerate Current Layout process is an option for regenerating the receiver and heliostat templates in memory without executing the heliostat field layout algorithm. If you have changed an input value and the values on the input pages no longer match the existing geometry that is stored in memory, the red warning symbol will appear next to the regenerate current layout button (shown above). When the specified input values match the solar field in memory, the warning icon will not appear.


Simulation Timer

- Top of page -


The simulation timer indicates SolarPILOT's progress in evaluating the simulation points and generating a new field layout. The Elapsed Time text indicates the total clock time that has elapsed since the simulation was invoked. The Simulation text indicates the number of simulations that have been completed and the total number of simulations that must be completed.


Simulation Log

- Top of page -


The simulation log prints messages that appear during the calculation process. These include status updates that note when a particular step in the layout process has been completed. The messages appear with a time stamp in milliseconds that indicate the time elapsed when the message was issued.


Field Layout

- Top of page -


The field layout group lists the heliostats in the field along with certain properties of interest. It also allows you to import or export custom layouts via text file.


Heliostat Count

The total number of heliostats in the layout.


Layout Import and Export

SolarPILOT allows the user to export and import field layouts. Exporting a field layout will generate information on the individual heliostat locations, focusing parameters, and aim points. The exported data is consistent with the data displayed on the Field Layout page in the layout dialog table.
