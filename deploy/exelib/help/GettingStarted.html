Formatting Conventions


This help menu uses several formatting conventions to aid the user in quickly scanning for relevant information.


The following conventions are used for Input Page References:



Variable Group


Input Item


Input Item Option or Case


Input Item Contingent Upon Another Input


This manual provides guidance and recommendations for choosing input values and settings. Dialog of this nature is formatted as follows:


Recommendation Dialog

Compass Conventions


SolarPILOT adopts the following convention regarding compass direction:


South ⇒ -180° = +180°

North ⇒ 0°

East ⇒ +90°

West ⇒ -90°
